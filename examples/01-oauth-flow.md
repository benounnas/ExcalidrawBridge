# OAuth 2.0 Authorization Code Flow

## Prompt

Draw the OAuth 2.0 Authorization Code flow. Show the User, Browser, App Server, and Auth Server as four vertical lanes. Walk through each step: the user clicks login, the browser redirects to the auth server, the user logs in and consents, the auth server redirects back with an authorization code, the app server exchanges the code for tokens, and finally the app server returns the user's profile. Use arrows between the lanes for each step, numbered 1 through 7. Use a light blue zone for the client side (User + Browser) and a light purple zone for the server side (App Server + Auth Server).
